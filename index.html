DrumMashN
Visually stimulating drum sample machine, user can select existing kits or create there own. Drum pads interact with the background spheres. The backend is built using rails and holds the sound sample information, users add new sound samples by providing links to the location of the hosted files. A user can also build a kit using the built in sounds. The front end is a threeJS canvas of spheres and a moving star universe.

Motivations and Inspirations

Will grew up wanting to be a hacker. When he was a little kid, he begged his Mum to buy him a c/c++ teach yourself at home course. Sending off his work eagerly every week and waiting for the results was an early lesson in development.
Coming from a mixed background in music education and the telecommunications engineering industry, Will has continued learning about software and hardware technology.
His real passion is in creation
"I love to create websites, images, applications, music, 3d worlds, posters, art, food, blogs, you name it, I'll create it!"
Will joined the Flatiron School in February 2019. There he learnt to build projects in Ruby/Javascript/React/Redux; contributing to his understanding of the full stack process and what it takes to be a professional developer. Since then he has continued learning and these days can be found reading Programming: Principles and Practice Using C++,
"it's not that I have ambitions of being a c++ developer, it's just that the book is so enlightening with regards programming. I recommend every developer should read it!!
I can't put it down."
